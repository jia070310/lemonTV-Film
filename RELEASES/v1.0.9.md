# v1.0.9 (10009)

**发布日期：** 2026-07-13

## 概述

本版本重点优化 TV 端存储与播放预缓存策略，降低磁盘占用与缓冲卡顿，并完善设置页缓存管理与确认弹窗体验。

## 体验改进

- **滚动预缓存**：HLS 仅预取播放头前方约 90~120 秒分片，播放中自动补充，不再全量下载占满磁盘
- **播放缓存分级上限**：按设备剩余空间限制为 64MB / 128MB / 192MB；退出播放或冷启动时自动清理
- **缓冲 UI 优化**：短暂缓冲不显示全屏「加载中」；已缓冲超过 5 秒时不遮挡画面
- **存储管理**：设置 → 播放设置新增播放/封面/首页数据缓存占用显示与分项清除
- **封面与首页缓存上限**：Coil 海报 48MB；首页 JSON 与观看历史封面按设备分级上限，满额 LRU 覆盖
- **确认弹窗**：清除缓存等确认框改为屏幕居中模态弹窗，不再嵌入底部控件栏

## 安装升级

1. 下载 `LomenTV-release-v1.0.9.apk`
2. 在电视 / 盒子上覆盖安装（数据与设置保留）
3. 若内置更新检测失败，可手动从 [Releases](https://github.com/jia070310/lemonTV-Film/releases) 下载安装

## 系统要求

- Android 5.0（API 21）及以上
- 可访问所配置的 MacCMS 站点

`HlsPrefetchService`、`PlayerDataSourceFactory`、`PlayerService`、`MediaStorageHelper`、`DeviceStorageProfile`、`HomeFeedDiskCache`、`WatchHistoryCoverStore`、`SettingsScreen`、`PlayerScreen`、`LomenTVApplication` 等。
