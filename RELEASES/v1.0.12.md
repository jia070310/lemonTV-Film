# v1.0.12 (10012)

**发布日期：** 2026-07-14

## 概述

本版本完善 MacCMS 播放项展示：详情页与播放器统一使用接口返回的集名/版本名，电影单版本（如 HD国语）也会直观显示，观看历史同步展示正确标签。

## 体验改进

- **播放版本展示**：电影即使只有一个版本（如 HD国语 / 粤语）也会在详情页显示「播放版本」
- **原始集名**：选集格子优先显示接口名（第1集、第01集、1、一集等），不再强行格式化为序号
- **电影多版本**：国语 / 粤语 / HD中字 等在详情页与播放器选集中正确显示
- **播放器选集**：单版本也可打开列表；切换集时使用统一标签格式
- **观看历史**：优先显示已保存的集名/版本名，而不是仅用数字集号

## 安装升级

1. 下载 `LomenTV-release-v1.0.12.apk`
2. 在电视 / 盒子上覆盖安装（数据与设置保留）
3. 若内置更新检测失败，可手动从 [Releases](https://github.com/jia070310/lemonTV-Film/releases) 下载安装

## 系统要求

- Android 5.0（API 21）及以上
- 可访问所配置的 MacCMS 站点

## 主要变更

`MacCmsPlayLayout`、`EpisodeLabelFormatter`、`DetailViewModel`、`DetailScreen`、`PlayerScreen`、`PlayerViewModel`、历史记录相关界面等。

## 构建命令

```bash
./gradlew :app:assembleRelease
```

输出：`app/build/outputs/apk/release/LomenTV-release-v1.0.12.apk`
