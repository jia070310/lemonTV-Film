# v1.0.10 (10010)

**发布日期：** 2026-07-14

## 概述

本版本修复预缓存进度条显示不准确问题，并优化详情页布局，减少空白区域，一屏内更容易看到选集。

## 体验改进

- **预缓存进度条**：缓冲条按实际预缓存时间点显示（约播放头前方 90~120 秒），不再误显示为整片已缓冲
- **详情页布局**：Hero 区域改为内容自适应高度，去除固定 500dp 留白
- **详情展开/收起**：简介、导演、演员默认收起；内容较多时可展开查看完整信息
- **简介清理**：去除 HTML 标签，避免 `<p>` 等脏文本占位
- **分类排序**：首页与筛选页分类顺序与 MacCMS 后台一致，修复排序颠倒问题

## 安装升级

1. 下载 `LomenTV-release-v1.0.10.apk`
2. 在电视 / 盒子上覆盖安装（数据与设置保留）
3. 若内置更新检测失败，可手动从 [Releases](https://github.com/jia070310/lemonTV-Film/releases) 下载安装

## 系统要求

- Android 5.0（API 21）及以上
- 可访问所配置的 MacCMS 站点

## 主要变更

`HlsPrefetchService`、`PlayerService`、`PlayerScreen`、`DetailScreen` 等。
