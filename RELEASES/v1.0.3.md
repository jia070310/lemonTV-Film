# v1.0.3 (10003)

**发布日期：** 2026-07-12

## 概述

本版本重点优化首页/筛选/详情页的加载体验，并增强播放器网络缓冲与 HLS 多线程预缓存能力。

## 新功能

- **卡片骨架加载**：首页与筛选页先显示卡片框架和片名，封面通过 `VodPosterImage` 逐张淡入
- **筛选页骨架网格**：切换筛选条件时显示骨架卡片，替代整页转圈
- **详情页秒开**：影片详情缓存 + 点击卡片写入快照，进入详情立即展示海报、标题与简介
- **HLS 多线程预缓存**：解析 m3u8 后 6 线程并行下载 TS 分片至本地缓存（参考 M3U8 下载工具模型）
- **播放缓冲可视化**：进度条显示浅灰缓冲条与 `预缓存 x/y` 分片计数

## 性能优化

- **播放器 OkHttp + 磁盘缓存**：播放网络栈改用 OkHttp 连接池，512MB `SimpleCache` 加速二次播放与 seek
- **影片详情内存缓存**：列表 enrich 与详情请求结果写入缓存，减少重复 `ac=detail` 请求
- **进度刷新 300ms**：播放器位置与预缓存状态更新更流畅

## 修复与体验改进

- **播放错误重试**：自动重试时保留 Cookie/Referer 请求头与字幕配置
- **详情页分步加载**：基本信息先展示，播放源与选集后台加载并显示进度提示

## 安装升级

1. 下载 `app-release.apk`
2. 在电视 / 盒子上覆盖安装（数据与设置保留）
3. 若内置更新检测失败，可手动从 [Releases](https://github.com/jia070310/lemonTV-Film/releases) 下载安装

## 系统要求

- Android 5.0（API 21）及以上
- 可访问所配置的 MacCMS 站点

## 完整变更文件

主要涉及：`VodPosterImage`、`HlsPrefetchService`、`PlayerDataSourceFactory`、`PlayerService`、详情/首页/筛选 UI 与 ViewModel、`MacCmsRepository` 详情缓存等。
