# v1.0.2 (10002)

**发布日期：** 2026-07-11

## 概述

本版本重点增强首页推荐展示、优化首页与筛选页加载速度，并全面改进筛选页遥控器焦点体验。

## 新功能

- **首页「最新推荐」**：在最近播放下方新增推荐区（MacCMS `level=9`），首页展示 11 张卡片 +「更多」入口
- **推荐列表页**：点击「更多」进入完整推荐列表，支持分页浏览
- **推荐数据兼容**：当站点 `ac=list&level=9` 接口未正确过滤时，自动改从模板页 `/vod/show/level/9.html` 解析并补全详情，确保推荐条目完整显示

## 性能优化

- **首页分阶段加载**：分类栏目先快速出列表，首屏可见卡片优先补封面，其余后台补全；推荐区与分类并行加载，互不阻塞
- **筛选页分阶段加载**：首屏仅拉取 20 条结果并优先补全可见封面，后台预取更多数据供「加载更多」使用
- **去除整页阻塞**：首页有内容即展示，仅在完全无数据时显示加载状态

## 修复与体验改进

- **筛选页焦点导航**
  - 打开页面默认焦点落在左上角分类栏
  - 从侧栏按 → 先进入顶部筛选条，再 ↓ 逐步下移，避免直接跳到卡片区
  - 卡片区支持 ← → 在卡片间平移
  - 修复翻到卡片区后按 ← 导致应用崩溃（`FocusRequester is not initialized`）
- **筛选页卡片左移**：修复卡片区按左键无法左移的问题

## 安装升级

1. 下载 `app-release.apk`
2. 在电视 / 盒子上覆盖安装（数据与设置保留）
3. 若内置更新检测失败，可手动从 [Releases](https://github.com/jia070310/lemonTV-Film/releases) 下载安装

## 系统要求

- Android 5.0（API 21）及以上
- 可访问所配置的 MacCMS 站点

## 完整变更文件

主要涉及：首页推荐 UI、推荐数据拉取、首页/筛选 ViewModel、筛选页焦点逻辑、导航路由等。
